# Code Style

- HTML: single-file app, inline `<style>` and `<script>` blocks
- CSS: use CSS custom properties for colors and spacing
- JS: vanilla ES6+, no frameworks, no build step
- Naming: camelCase for JS, kebab-case for CSS classes
- Keep functions small — extract helpers when logic exceeds 30 lines
- Comments only where intent isn't obvious from the code
