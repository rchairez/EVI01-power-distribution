# /project:deploy

Deploy the power distribution app to GitHub Pages.

## Steps
1. Run a quick review pass (no hardcoded secrets, no broken references)
2. Ensure proxies are not required for static deploy (or document the limitation)
3. Commit all changes with a descriptive message
4. Push to main branch
5. Verify GitHub Pages is enabled (Settings → Pages → main branch)
6. Report the live URL: `https://rchairez.github.io/EVI01-power-distribution/`
